HomeWork Hadoop-Mapreduce

Adapaun cara untuk menjalankan library mapreduce pada python, sebagai berukut:

-Pertama yaitu istall terlebih dahulu libary mapreduce pada visual studio code, dengan menuliskan perintah "pip install mapreduce" pada terminal visual studio code.
-Selanjutnya buat file python pada VSCode "lib_mapreduce.py"
-Kemudian siapkan satu file .txt yang akan di count datanya. Dalam tugas sekarang data yang di count yaitu data benda "benda.txt".
-Masukan source code bawaan pada python untuk menjalankan fungsi hadoop-mapreduce
-Untuk running program dapat menulisakan "python lib_mapreduce.py benda.py" pada terminal, kemudian enter. Maka data akan muncul beserta jumlahnya.
